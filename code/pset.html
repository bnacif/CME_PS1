Estimating the number of players (single tickets)

For each game \(j\), we estimate the number of participants (\(N_j\)) using the formula: \[ N_j = \frac{revenue}{price} \] Note that we assume that all individuals choose the simplest and cheapest option. This a fairly strong assumption given that players can choose among several options of entering the lottery game. Still, this gives us a rather good aproximation to start with.

where \(G\theta_j\) is the number of individuals who won one of the prizes, depending on the number \(\theta \in \{4,5,6\}\) of numbers they got right, and \(Y_j^{\theta}\) is the corresponding prize paid to each winner.
